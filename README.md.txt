### API Endpoints

#### GET
  - uri: https://bibek-tws.herokuapp.com/api/tws
  
#### POST
  - uri: https://bibek-tws.herokuapp.com/api/tws
  - body: { "message": "Your message" }
  
#### GET
  - uri: https://bibek-tws.herokuapp.com/api/tws/id
  - params: id (MongoDB Document Object ID)

#### PATCH
  - uri: https://bibek-tws.herokuapp.com/api/tws/id
  - params: id (MongoDB Document Object ID)
  - body: { "message": "Your message" }

#### DELETE
  - uri: https://bibek-tws.herokuapp.com/api/tws/id
  - params: id (MongoDB Document Object ID)
